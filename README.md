# Aerospace.js

Aerospace.js is a typescript/javascript library that will compute gas dynamics problems for you

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install aerospace.js in your project

```
npm install aerospace
```

## Usage

This project is still in the planning stage of development, and should not be consumed yet

## Development setup

1. Clone this project: ``git clone git@github.com:samrauer/aerospace.git``

2. Navigate into directory: ``cd aerospace``

3. Install dev dependencies: ``npm install``

## Scripts

- Build the project: ``npm run build``
- Test the project: ``npm run test``
- Publish the package: ``npm publish``
